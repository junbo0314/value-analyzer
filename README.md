# Value Analyzer

Graham-Dodd 내재가치 공식을 기반으로 미국·한국 주식의 내재가치(IV)를 계산하고 현재 주가와 비교해주는 웹 앱입니다.

**Live Demo →** _배포 후 URL 추가_

---

## 어떤 앱인가요?

티커를 입력하면 아래 항목을 자동으로 계산합니다.

- **내재가치 (Intrinsic Value)** — Graham-Dodd 공식 기반
- **현재가 대비 괴리율** — 저평가/고평가 판단
- **5단계 공식 계산 상세** — 각 변수가 IV에 미치는 영향 시각화
- **AI 신호** — FCF 경고, 베타 경고, 성장 긍정 신호 등

## 공식

```
IV = (EPS × [BasePER + (gMult × g)] × CP × ROE_Factor × 4.0 ÷ Y) × DF + 순현금/주
```

| 변수 | 의미 | 결정 방식 |
|------|------|----------|
| EPS | 주당순이익 | Yahoo Finance Forward EPS |
| BasePER | 기준 PER | 등급별 (4.5 ~ 14.0) |
| gMult | 성장률 가중치 | 등급별 (1.0 ~ 2.25) |
| g | 예상 성장률 | EPS CAGR / 매출성장률 (슬라이더 조정 가능) |
| CP | 경쟁력 프리미엄 | 매출총이익률 기반 0.80~1.20 |
| ROE_Factor | 자본효율성 승수 | ROE 기반 0.90~1.10 |
| Y | AAA 채권 금리 | 10Y 국채 + 0.5% 실시간 |
| DF | 부채 페널티 | 부채비율 기반 0.70~1.00 |

## 등급 시스템

기업의 경제적 해자(moat)를 4단계로 분류합니다. 사용자가 직접 선택합니다.

| 등급 | 명칭 | BasePER | gMult | 예시 |
|------|------|---------|-------|------|
| 1등급 | 압도적 지배자 | 14.0 | 2.25 | MSFT, GOOGL |
| 2등급 | 강력한 과점 주도자 | 13.0 | 1.50 | META, NVDA, AAPL |
| 3등급 | 유력한 경쟁자 | 7.0 | 1.00 | AMD, 삼성전자 |
| 4등급 | 취약한 추격자 | 4.5 | 1.00 | 소형주, 하청 제조 |

> 파라미터는 SEC EDGAR 10년 데이터(2010–2026, S&P 500 319개 종목)로 백테스팅한 결과를 반영했습니다.

## 백테스팅 결과 요약

319개 S&P 500 종목 / 2010–2026년 / 2,351개 연간 데이터 포인트

| 구분 | Spearman r | Q5−Q1 수익률 차이 |
|------|-----------|----------------|
| Train 2010–2017 | −0.044 | — |
| **Val 2018–2026** | **+0.079** (p<0.05) | **+8.4%p** |

- Q5 (IV/Price 상위 20%) 평균 1년 수익률: **+21.4%**
- Q1 (IV/Price 하위 20%) 평균 1년 수익률: **+13.0%**
- 상위 3개 아웃라이어 제거 후에도 스프레드 거의 동일 (+8.5%p)

## 로컬 실행

```bash
# 1. 의존성 설치
npm install

# 2. 개발 서버 실행
npm run dev

# 3. http://localhost:3000 접속
```

Node.js 18+ 권장.

외부 API 키 없이 실행됩니다. 주식 데이터는 Yahoo Finance에서 실시간으로 가져옵니다.

## 기술 스택

- **Frontend / Backend**: Next.js 14 (App Router), TypeScript, Tailwind CSS
- **데이터**: yahoo-finance2 v3
- **배포**: Vercel

## 주의사항

이 앱은 투자 참고용 정량 도구입니다. 투자 결정의 유일한 근거로 사용하지 마세요.

- IV/Price가 높다고 반드시 오르는 것은 아닙니다 (Q5 내에서도 큰 손실 사례 다수)
- 등급 배정(1~4등급)은 사용자의 주관적 판단에 의존합니다
- 공식이 고ROE·고마진 기업(MSFT 등)에게 구조적으로 높은 IV를 산출하는 경향이 있습니다
